# Random Task Generator

**Автор:** Данил Симбирцев

## Описание
GUI-приложение для генерации случайных задач.  
Поддерживает:
- случайный выбор задачи;
- историю генерации;
- фильтрацию по типу;
- сохранение и загрузку истории в JSON;
- добавление новых задач с проверкой ввода.

## Запуск
1. Убедитесь, что установлен Python 3.
2. Запустите файл:
   python main.py

## Примеры использования
- Нажмите **Сгенерировать задачу**.
- Выберите фильтр: учёба / спорт / работа.
- Добавьте новую задачу и проверьте, что пустая строка не принимается.
- Закройте программу и откройте снова — история загрузится из JSON.
